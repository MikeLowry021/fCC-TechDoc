Introduction to HTML

HTML (HyperText Markup Language) is the foundation of web development, providing the structure for content on the internet.

It uses tags to define elements like headings, paragraphs, and links.

<html> <head> <title>My Page</title> </head> <body> <h1>Welcome</h1> </body> </html>
  • HTML documents start with a <!DOCTYPE html> declaration.
  • The <html> tag is the root element.
CSS Fundamentals

CSS (Cascading Style Sheets) controls the visual presentation of HTML elements.

It allows you to style colors, fonts, layouts, and more.

selector { property: value; }
  • CSS rules consist of selectors and declarations.
  • Declarations are made up of properties and values.
HTML Structure

HTML documents are organized into a tree-like structure called the DOM (Document Object Model).

Common elements include <div>, <section>, and <article> for grouping content.

<section> <h2>Section Title</h2> <p>Content here</p> </section>
CSS Styling Techniques

CSS can be applied inline, internally, or externally via a stylesheet.

External stylesheets are preferred for maintainability.

<link rel="stylesheet" href="styles.css">

You can target elements using classes, IDs, or tag names.

.class-name { color: blue; } #id-name { font-size: 16px; }
Responsive Design

Responsive design ensures websites adapt to different screen sizes.

Media queries and flexible units like percentages or rem/em are key.

  • Use relative units for scalability.
